# Turbo Repository Health Check

Perform a comprehensive check of the Turbo repository.

1. Run `pnpm lint` and fix any linting issues
2. Run `pnpm test` and fix any failing tests
3. Run `pnpm build` and fix any build errors
4. Check for outdated dependencies with `pnpm outdated`
5. Verify all packages follow the workspace conventions
6. Ensure turbo.json is properly configured
7. Provide a health report summary
